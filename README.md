# vue-core-template
一个自定义的功能完备的vue3模板

## 模板目录：
```
│  .gitignore
│  index.html
│  LICENSE
│  package-lock.json
│  package.json
│  README.md
│  tsconfig.json
│  vite.config.ts
│
├─.vscode
│      extensions.json
│
├─public
│      favicon.ico
│
└─src
    │  App.vue
    │  env.d.ts
    │  main.ts
    │
    ├─assets
    │      logo.png
    │
    ├─components
    ├─pages
    │  ├─home
    │  │      home.vue
    │  │
    │  └─login
    │          login.vue
    │
    ├─router
    │      index.ts
    │
    ├─service
    ├─store
    │      home.ts
    │      index.ts
    │      login.ts
    │
    └─utils
```

## 模板已含有的内容：
- vue3(vue-core) + typescript支持
- vite
- router4(The official router for Vue.js)
- pinia
- directory alias('@/')

## 设计目的：
作为自主开发脚手架的vue3模板
